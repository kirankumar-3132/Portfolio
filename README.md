# Personal Portfolio Website

## Project Overview
This is a responsive personal portfolio website built using HTML, CSS, and JavaScript.  
The website presents personal information, skills, interests, and contact details with smooth animations and an interactive image gallery.

---

## Technologies Used
- HTML5
- CSS3
- JavaScript

---

## Project Structure
portfolio/
├── index.html
├── style.css
├── script.js
└── images/

---

## Features

### Sticky Header with Navigation
- Header remains visible while scrolling
- Navigation sections:
  - About
  - Skills
  - Gallery
  - Contact

---

### Animated Hero Section
- Text reveal animation
- Floating background shapes
- Smooth, non-distracting transitions

---

### Personal Information Section
- Name, role, and brief introduction
- Click-to-toggle illustrated avatar
- Quick facts (languages, interests, focus)
- Resume download button (placeholder)

---

### Skills Section
- Displays technical skills using pill-style tags
- Clean and readable layout

---

### Interactive Image Gallery
- Shows hobbies and inspirations
- Includes:
  - Watching Cricket
  - MS Dhoni
  - Pawan Kalyan
- Hover zoom interaction on images
- Images embedded using Base64

---

### Contact Section
- Contact form with:
  - Name
  - Email
  - Message
- Submit button
- Responsive layout

---

### Responsive Design
- Works on desktop, tablet, and mobile devices
- Uses Flexbox and media queries

---

## How to Run
1. Download or clone the project
2. Open `index.html` in any web browser
